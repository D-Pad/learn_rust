Infallable Type

Infallable is an enum with no ways to instantiate it, which isn't ordinarily allowed. It is sometimes used as the return value of a function that can never return. 

pub enum Infallible {}

Useful for functions that run in an infinite loop, like an HTTP server. 

fn t_1000() -> Infallible {
    loop {
        seek_humans();
        kill_humans();
    }
}
